أهم 5 ويدجتات في Flutter للمبتدئين

شرح تفصيلي مع أمثلة عملية ونصائح أداء

1. Text Widget

ما هي ويدجت Text؟

أبسط ويدجت في Flutter لعرض النصوص بكل أنواع التنسيقات. تدعم العربية وتتحكم في كل شيء من حجم الخط إلى الظلال!

مثال عملي

Text(
  'مرحباً بكم في Flutter',
  style: TextStyle(
    fontSize: 24,
    fontWeight: FontWeight.bold,
    color: Colors.blue,
    fontFamily: 'Tajawal',
    decoration: TextDecoration.underline,
  ),
  textAlign: TextAlign.center,
  maxLines: 2,
  overflow: TextOverflow.ellipsis,
)

معاينة النص

مرحباً بكم في Flutter

نصائح ذهبية

النصوص الطويلة

استخدم maxLines و overflow للتحكم في النصوص التي تتجاوز المساحة.

النصوص المخصصة

استخدم Text.rich لعمل نصوص بأجزاء مختلفة الأنماط.

الأداء

تجنب تغيير أنماط النص بشكل متكرر في setState.

2. Image Widget

ما هي ويدجت Image؟

الويدجت المسؤولة عن عرض الصور في تطبيقك من مصادر مختلفة: الإنترنت، الملفات المحلية، أو الذاكرة.

طرق تحميل الصور

// من الإنترنت
Image.network('https://example.com/image.jpg')

// من ملفات المشروع
Image.asset('assets/images/logo.png')

// من الذاكرة (بايتس)
Image.memory(bytes)

خصائص متقدمة

Image.network(
  'https://example.com/photo.jpg',
  width: 200,
  height: 150,
  fit: BoxFit.cover,
  loadingBuilder: (context, child, progress) {
    if (progress == null) return child;
    return CircularProgressIndicator();
  },
  errorBuilder: (context, error, stackTrace) {
    return Icon(Icons.error);
  },
)
معلومة تقنية

استخدم حزمة cached_network_image لتحسين أداء الصور من الإنترنت:

dependencies:
  cached_network_image: ^3.2.0

أنواع BoxFit

النوع الوصف
BoxFit.fill تملأ الصورة المساحة مع تجاهل النسب
BoxFit.cover تحافظ على النسب وتغطي المساحة (قد تقطع جزءاً)
BoxFit.contain تحافظ على النسب وتظهر الصورة كاملة

3. Container Widget

ما هي ويدجت Container؟

صندوق متعدد الاستخدامات يمكنك التحكم في شكله، حجمه، خلفيته، هوامشه، وظله. يعتبر من أكثر الويدجتات استخداماً.

مثال شامل

Container(
  width: 300,
  height: 200,
  margin: EdgeInsets.all(20),
  padding: EdgeInsets.symmetric(vertical: 10, horizontal: 15),
  decoration: BoxDecoration(
    color: Colors.blue[100],
    borderRadius: BorderRadius.circular(15),
    border: Border.all(
      color: Colors.blue,
      width: 2,
    ),
    boxShadow: [
      BoxShadow(
        color: Colors.grey.withOpacity(0.5),
        blurRadius: 10,
        offset: Offset(0, 5),
      ),
    ],
  ),
  child: Text('أنا داخل الكونتينر!'),
)

معاينة الكونتينر

أنا داخل الكونتينر!

استخدامات شائعة

البطاقات

إنشاء بطاقات بتصميمات جذابة مع ظلال وزوايا مدورة.

الخلفيات

إضافة خلفيات مخصصة لأجزاء من الشاشة.

الحدود

رسم حدود حول العناصر مع تخصيص السمك واللون.

4. Row & Column

ما الفرق بينهما؟

Row Column
يرتب العناصر أفقيًا يرتب العناصر رأسيًا
mainAxisAlignment يتحكم في المحور الأفقي mainAxisAlignment يتحكم في المحور الرأسي
crossAxisAlignment يتحكم في المحور الرأسي crossAxisAlignment يتحكم في المحور الأفقي

أمثلة عملية

// مثال Row
Row(
  mainAxisAlignment: MainAxisAlignment.spaceBetween,
  children: [
    Icon(Icons.star),
    Text('عنصر 1'),
    Container(width: 50, height: 50, color: Colors.red),
  ],
)
// مثال Column
Column(
  crossAxisAlignment: CrossAxisAlignment.start,
  children: [
    Text('الاسم: أحمد'),
    Text('العمر: 25'),
    Text('البلد: مصر'),
  ],
)

معاينة الترتيب

عنصر 1
الاسم: أحمد
العمر: 25
البلد: مصر

نصائح متقدمة

دمج Row و Column

يمكنك وضع Column داخل Row والعكس لإنشاء تخطيطات معقدة:

Column(
  children: [
    Row(children: [/*...*/]),
    Container(/*...*/),
    Column(children: [/*...*/]),
  ],
)

5. Padding Widget

ما هي ويدجت Padding؟

تضيف مسافة داخلية حول العنصر (بين محتوى العنصر وحوافه). تختلف عن margin التي تضيف مسافة خارجية.

طرق تطبيق Padding

// جميع الجهات بنفس القيمة
Padding(
  padding: EdgeInsets.all(20),
  child: Text('أنا محاط بمسافة 20 بكسل'),
)

// محور أفقي/رأسي مختلف
Padding(
  padding: EdgeInsets.symmetric(vertical: 10, horizontal: 15),
  child: Text('أعلى/أسفل: 10 - يمين/يسار: 15'),
)

// تخصيص كل جهة
Padding(
  padding: EdgeInsets.only(top: 10, left: 20, bottom: 5),
  child: Text('تخصيص لكل جهة'),
)

الفرق بين Padding و Margin

Padding داخلي
Margin خارجي

متى نستخدم Padding؟

منع التصاق النص

إضافة مسافة بين النص وحواف العنصر المحيط.

تحسين القراءة

جعل المحتوى أكثر راحة للعين بإضافة مسافات مناسبة.

التنسيق الداخلي

إضافة مسافة بين العناصر الداخلية بدون استخدام SizedBox.
