1️⃣ Text Widget

تُستخدم لعرض النصوص داخل التطبيق. يمكنك التحكم في تنسيق النصوص مثل اللون، الحجم، والمحاذاة.

أهم الخصائص:

  • textAlign: تحديد محاذاة النص (يسار، يمين، مركز).
  • style: تنسيق النص (حجم الخط، اللون، الوزن، العائلة).
  • maxLines: تحديد أقصى عدد للأسطر.
  • overflow: كيفية التعامل مع النصوص الطويلة.

Text(
  'مرحبًا بك في فلاتر!',
  textAlign: TextAlign.center,
  style: TextStyle(
    fontSize: 24, 
    fontWeight: FontWeight.bold, 
    color: Colors.blue
  ),
  maxLines: 1,
  overflow: TextOverflow.ellipsis,
)

2️⃣ Image Widget

تُستخدم لعرض الصور سواء من الملفات المحلية أو من الإنترنت. يمكنك التحكم في حجم الصورة وكيفية ملء المساحة المتاحة.

أهم الخصائص:

  • image: تحديد مصدر الصورة (من الإنترنت أو من الملفات المحلية).
  • width/height: تحديد أبعاد الصورة.
  • fit: التحكم في كيفية ملء الصورة للحاوية.
  • alignment: تحديد محاذاة الصورة داخل الحاوية.

Image.asset(
  'assets/images/flutter_logo.png', 
  width: 100, 
  height: 100, 
  fit: BoxFit.cover
)

        

Image.network(
  'https://flutter.dev/assets/images/shared/brand/flutter-lockup.png', 
  width: 100, 
  height: 100,
  fit: BoxFit.contain
)

3️⃣ Container Widget

تُستخدم كعنصر حاوي لتنظيم العناصر الأخرى. يمكنك التحكم في الأبعاد، الألوان، الحدود، الهوامش، والمسافات الداخلية.

أهم الخصائص:

  • width/height: تحديد أبعاد الحاوية.
  • color: تحديد لون الخلفية.
  • padding: إضافة مسافات داخلية.
  • margin: إضافة مسافات خارجية.
  • decoration: تخصيص الخلفية والحدود.

Container(
  width: 200, 
  height: 100, 
  padding: EdgeInsets.all(10),
  margin: EdgeInsets.symmetric(horizontal: 20),
  decoration: BoxDecoration(
    color: Colors.blue, 
    borderRadius: BorderRadius.circular(15),
    boxShadow: [BoxShadow(color: Colors.black26, blurRadius: 10)],
  ),
  child: Center(
    child: Text(
      'أنا كونتينر!',
      style: TextStyle(color: Colors.white, fontSize: 18),
    ),
  ),
)

4️⃣ Row & Column Widgets

تُستخدم لتنظيم العناصر بشكل أفقي (Row) أو عمودي (Column).

أهم الخصائص:

  • mainAxisAlignment: تحديد محاذاة العناصر في الاتجاه الرئيسي (أفقي/عمودي).
  • crossAxisAlignment: تحديد المحاذاة العمودية (يسار، يمين، وسط).

Row(
  mainAxisAlignment: MainAxisAlignment.spaceEvenly,
  children: [
    Icon(Icons.star, color: Colors.yellow, size: 30),
    Icon(Icons.star, color: Colors.yellow, size: 30),
    Icon(Icons.star, color: Colors.yellow, size: 30),
  ]
)

        

Column(
  crossAxisAlignment: CrossAxisAlignment.start,
  children: [
    Text('العنصر الأول'),
    SizedBox(height: 10),
    Text('العنصر الثاني'),
  ]
)

5️⃣ Padding Widget

تُستخدم لإضافة مسافات داخلية حول العناصر لتجنب التكدس.

أهم الخصائص:

  • padding: تحديد المسافات الداخلية حول العنصر.
  • child: العنصر الذي سيتم إضافة Padding حوله.

Padding(
  padding: EdgeInsets.all(16.0),
  child: Text(
    'هذا نص داخل ويدجت Padding!', 
    style: TextStyle(fontSize: 20)
  )
)
